Write article about cryptocurrencies

This editor helps you write articles about cryptocurrencies. You can easly get particular cryptocurrency's name and it's current price.

{{ Name/currencySymbol }}
use this pattern to get curreny's name. For examble {{ Name/BTC }}

{{ Price/currencySymbol }}
use this pattern to get the currency's current price in USD. For examble {{ Price/BTC }}

Example text

"In October 2011, {{ Name/LTC }} ({{ Price/LTC }}) was released. It was the first successful cryptocurrency to use scrypt as its hash function instead of SHA-256."
